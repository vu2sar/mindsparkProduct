    Score Calculation

    • + Weight of Accuracy X Score of accuracy - Accuracy is the percentage of questions attempted correctly. Minimum questions, as per school requirement, need to be attempted per week
    • + Weight of Badges X Score of Badges - Badges are given monthly to students in Mindspark for exceptional performance. Each badge gives 10 points out of 100
    • + Weight of Weekly Usage X Score of Weekly Usage - Weekly usage score is the percentage of weeks when usage exceeded the minimum weekly usage requirement
    • + Weight of Topic Completion X Score of Topic Completion - Topic completion score is the percentage of topics where progress exceeded 75%, out of those activated by teacher in the report period

    What is a weighted score?
    Weighted score is calculated by "weighing" the score of each parameter and adding them up. For example if the weight of "Accuracy" is 40% and student scores 100, the score added will be 40%*100 = 40. Similar calculations will be done for other parameters and added to the weighted score
    Why do we use a weighted score?
    A student can have many achievements on Mindspark. Some may attempt questions accurately, some may complete all topics with diligence, and so on. We thus have more than one parameter on which a student's achievement needs to be judged. To provide a single score finally, all the individual parameter scores need to be weighted. These weights can be modified by the school
    What parameters have been chosen and why?
    Mindspark performance report seeks to measure the proficiency as well as diligence of the student. For measuring proficiency, two parameters - accuracy and badges earned has been chosen. For measuring diligence, two parameters - weekly usage and topic completion have been chosen.

    The weighted score is arrived at as (accuracy score x accuracy weight + badges score x badges weight + Topic completion score x Topic Completion weight + Weekly usage score x Weekly usage weight)

    How can I use this score?
    This score is indicative of the student's performance on Mindspark, the parameters where the student is doing relatively better and parameters which require improvement. Usage of this score by the school is as per discretion of the school.
